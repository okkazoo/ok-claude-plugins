# OK Claude Plugins

A collection of plugins for Claude Code by OKKazoo.

## Installation

```bash
# Add marketplace (one-time)
/plugins marketplace add okkazoo/ok-claude-plugins

# Install a plugin
/plugins install <plugin-name>
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [echo](./echo/) | Session intelligence with activity tracking, structure capture, and worklog summaries |
| [ok-know](./ok-know/) | Knowledge management with WIP tracking, save points, and auto-indexing |
